Offshore Wind Portfolio Optimization
Data Citations and Source Notes
Prepared: March 29, 2026

This file lists the external data sources used to generate the analysis,
derived data products, figures, and interactive map in this repository.

Repository outputs derived from these sources include:
- `data/bathymetry_processed.parquet`
- `data/era5_month_01.parquet` through `data/era5_month_12.parquet`
- `data/era5_annual_means_2025.parquet`
- `data/suitability_results_2025.parquet`
- `data/prime_wind_sites.parquet`
- `data/map_data.js`
- all figures in `outputs/plots/`

1. ERA5 atmospheric, land-surface, and ocean-wave reanalysis

Primary citation:
Copernicus Climate Change Service (C3S), implemented by ECMWF. ERA5 hourly
data on single levels from 1940 to present. Climate Data Store.
DOI: 10.24381/cds.adbb2d47

Official sources:
https://cds.climate.copernicus.eu/datasets/reanalysis-era5-single-levels?tab=download
https://www.ecmwf.int/en/forecasts/dataset/ecmwf-reanalysis-v5

Used in this project for:
- 100 m wind vectors (`u100`, `v100`)
- 2 m temperature (`t2m`)
- surface pressure (`sp`)
- land-sea mask (`lsm`)
- significant wave height (`swh`)
- mean wave period (`mwp`)
- monthly and annual offshore wind power density calculations for 2025

Local code references:
- `02_wind_era5.py`
- presentation references to "ERA5 Reanalysis"

2. GEBCO 2025 bathymetry grid

Primary citation:
GEBCO Compilation Group (2025). GEBCO 2025 Grid.
DOI: 10.5285/37c52e96-24ea-67ce-e063-7086abc05f29

Official source:
https://www.gebco.net/data-products-gridded-bathymetry-data/gebco2025-grid

Used in this project for:
- ocean depth / bathymetry
- land masking during bathymetry preprocessing
- slope estimation
- foundation-type classification
- bathymetry figures and suitability screening

Local code references:
- `01_eda.py`
- presentation references to "GEBCO 2025 Bathymetry"

3. BOEM renewable energy lease and planning area reference data

Source organization:
U.S. Department of the Interior, Bureau of Ocean Energy Management (BOEM)

Official source:
https://www.boem.gov/renewable-energy/mapping-and-data/renewable-energy-gis-data

Used in this project for:
- qualitative / visual validation against BOEM offshore wind lease areas
- naming and approximate placement of lease / Wind Energy Area overlays in the
  interactive map
- validation slide claiming alignment with BOEM lease areas

Important note:
The BOEM lease polygons shown in `interactive_map.html` are approximate
bounding overlays manually encoded for presentation purposes. They are based on
official BOEM lease / planning-area references, but they are not direct
downloads of BOEM GIS geometries.

Local code references:
- `interactive_map.html`
- BOEM validation slides in `datathon_presentation.html`

4. U.S. state boundary basemap for the interactive map

Basemap file used locally:
- `data/states-10m.js`

Source citation:
`topojson/us-atlas` - pre-built TopoJSON from the U.S. Census Bureau.
The `states-10m.json` topology is described by the project as being derived
from U.S. Census Bureau cartographic state boundaries (2017 edition).

Official source:
https://github.com/topojson/us-atlas

Used in this project for:
- drawing U.S. state outlines in the interactive portfolio map

5. Project-derived data products

The following are project-generated outputs and are not separate external
datasets:
- `data/map_data.js`
- all parquet files generated by the analysis scripts
- all figures in `outputs/plots/`

These files were derived from the ERA5 and GEBCO inputs above, plus project
logic for filtering, clustering, optimization, and presentation formatting.
